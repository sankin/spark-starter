# spark-starter

This example demonstrate the difference in performance between `reduceByKey` and `groupBy` operations when counting simple key-value pairs.

## Prerequisites

* Maven 3
* Java 8

## To Build
`mvn clean install`

## To Run

#### Maven
`mvn spring-boot:run`

#### Java

`java -jar target/spark-starter-0.0.1-SNAPSHOT.jar`
